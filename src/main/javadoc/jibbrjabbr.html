What Is JibbrJabbr?

It's an experimental HTTP server with a few neat pieces. It runs on the JVM, exposes a JavaScript API, and offers fairly good resource management. It's fully asynchronous, made as effortless as I am able. The Java API works via tasks and events. The JavaScript API uses Rhino's continuation feature to expose a synchronous program flow while remaining fully event driven underneath.
